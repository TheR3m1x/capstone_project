Core Tienda

Resumen del proyecto

En esta proyecto lo que se realizo fue un Core con las partes necesarias para una empresa.

Tecnologias Usadas:

  • C#
  • Windows Forms
  • SQL Server
  • Web Services

Funcionalidades

  • Niveles de Acceso por rol de Usuario
    1. Administrador
    2. Mantenimiento
    3. Usuarios
    4. Consultas
  • Contraseña compleja
    • Minimo 8 digitos
    • Una Mayuscula
    • Una Minuscula
    • Un Numero
    • Un Caracter
  • Poder Buscar dentro de las Tablas
  • Poder hacer todo lo basico de un CRUD
  • Validaciones dentro de la base de datos
